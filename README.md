#  Stopwatch Web Application

A modern stopwatch web application built using **HTML**, **CSS**, and **JavaScript** as part of the **Prodigy InfoTech Web Development Internship (Task 02)**.

# Features

- Start Timer
- Pause Timer
- Stop Timer
- Reset Timer
- Lap Recording
- Lap History
- Mission Control Dashboard
- Fastest Lap
- Slowest Lap
- Total Lap Counter
- Current Time Display
- Animated Glowing Timer Ring
- Twinkling Star Background
- Modern Responsive UI

# Technologies Used

- HTML5
- CSS3
- JavaScript
